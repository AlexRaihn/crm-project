My graduation project.

It is a template for developing various information systems.

The project includes the following:
- Authorization
- Data warehouses and their filling with mock data
- basic structure
- main functions helpers: HTTP client factory, tryCatch, localFilter, DialogFunction, DataFormatter
- Private page template and their protection from unauthorized users

What is planned for the future?
- adding completely custom UI components
- improvement of the function useDialog
- full style customization
- work with localization
